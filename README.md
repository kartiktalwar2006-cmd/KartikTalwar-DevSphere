# DevSphere

DevSphere is a full-stack developer community platform where developers can create professional profiles, share posts, connect with other developers, and showcase their experience, education, and GitHub projects. The platform combines social networking with professional development, helping developers build their online presence and collaborate with the tech community.

## 🚀 Features

* Secure JWT authentication and authorization
* Developer profile management
* Add experience and education details
* Create, edit, and delete posts
* Like and comment on posts
* Browse developer profiles
* GitHub repository integration
* Responsive design for all devices
* Protected routes and dashboard

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux
* React Router
* Axios
* Bootstrap

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## ⚙️ Installation

```bash
git clone https://github.com/kartiktalwar2006-cmd/DevSphere.git
cd DevSphere
```

Install dependencies:

```bash
cd client && npm install
cd ../ && npm install
```

Run the project:

```bash
# Frontend
npm run client

# Backend
npm run server

# Both
npm run dev
```
